## 技術

- CSS/ Bootstrap v5.1.3:rwd,排版
- laravel:blade,route,model,controller技術 會員權限控制進入會員中心後臺管理
- JS/ jQuery 3.7.1
- MySQL 4.1.3:CRUD,正規化,JOIN
- JS/ sweetalert2
- leaflet
- vue:vue組件
- js/chart.js
- js/counterup.js
